# Fuente New Sun Playful

Coloca aquí el archivo de la fuente **New Sun Playful** para que se cargue en toda la web.

**Nombre del archivo** (uno de estos):
- `NewSunPlayful.woff2` (recomendado)
- `NewSunPlayful.woff`
- `NewSunPlayful.ttf`

Si el archivo tiene otro nombre (por ejemplo con espacios), renómbralo a `NewSunPlayful` con la extensión correspondiente.

Puedes descargar la fuente desde sitios como Envato Elements (licencia comercial) o freefontsfamily.org (solo uso personal). Para uso web conviene convertir TTF/OTF a WOFF2 con [CloudConvert](https://cloudconvert.com/ttf-to-woff2) o [Transfonter](https://transfonter.org/).
